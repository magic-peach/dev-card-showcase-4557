# Emotional Support Network

A platform connecting users anonymously for peer-to-peer emotional support. Features include chat rooms, one-on-one sessions, moderation tools, mental health resources, and user matching.

## Features
- Anonymous chat rooms
- One-on-one support sessions
- Moderation tools
- Mental health resources
- User matching by needs/topics

## Advanced Features
- Modular chat rooms and one-on-one sessions
- User authentication (anonymous login)
- Moderation with report review and resolution
- Mental health resources (expandable)
- User matching with history tracking
- Privacy-first design (no personal data stored)

## Folder Structure
- `assets/` — images, icons, and static files
- `components/` — reusable UI components
- `utils/` — helper functions and utilities
- `app.js` — main application logic
- `index.html` — main UI
- `style.css` — global styles

## Getting Started
Open `index.html` in your browser to start using the platform.

## Contribution
Feel free to contribute by improving features, UI, or adding new resources.

## How to Use
1. Open `index.html` in your browser.
2. Use navigation to access chat rooms, one-on-one sessions, moderation, resources, and matching.
3. Log in anonymously to join chats.
4. Report messages and resolve reports as moderator.
5. Match with peers based on topics and view match history.

## Development
- Components are in `/components`
- Utilities are in `/utils`
