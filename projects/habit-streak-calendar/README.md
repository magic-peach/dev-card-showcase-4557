# Habit Streak Calendar

Mark daily habits and visualize streaks on a calendar.

## Features
- Add, edit, and delete habits
- Mark habits as done for each day
- Visualize streaks on a calendar
- View stats (longest streak, current streak, completion rate)
- Settings for customization
- Data persists in local storage

## How to Use
1. Open `index.html` in your browser.
2. Add your habits and mark them daily.
3. View your streaks and stats.
