# Local Event Finder

Discover and bookmark local events, workshops, or meetups.

## Features
- Search for local events by location, date, or keyword
- Bookmark favorite events
- Beautiful, modern UI
- Settings for customization
- Data persists in local storage

## How to Use
1. Open `index.html` in your browser.
2. Search for events and bookmark your favorites.

## Future Improvements
- Integrate with external event APIs
- Export/import data
- Mobile-friendly design
