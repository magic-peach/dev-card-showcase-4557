# Interactive Language Pronunciation Coach

Practice speaking new languages with real-time voice feedback and pronunciation scoring. Includes progress tracking.

## Features
- Language selection
- Practice words & phrases
- Real-time speech recognition
- Pronunciation scoring
- Progress tracking

## Setup
Open `index.html` in your browser. Requires browser with SpeechRecognition API support.
