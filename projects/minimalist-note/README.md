# Minimalist Note

Quick notes with tags, search, and simple formatting.

## Features
- Create, edit, and delete notes
- Tag notes and filter by tag
- Search notes by content or tag
- Simple formatting (bold, italic, code, lists)
- Beautiful, minimalist UI
- Settings for customization
- Data persists in local storage

## How to Use
1. Open `index.html` in your browser.
2. Create notes, add tags, and search your notes.

## Future Improvements
- Export/import notes
- Mobile-friendly design
- Markdown support
