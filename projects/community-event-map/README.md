# Local Community Event Map & RSVP Platform

Browse, create, and RSVP to local events on an interactive map. Includes notifications, event details, and social features.

## Features
- Interactive map (Leaflet.js)
- Event creation and browsing
- RSVP functionality
- Attendee tracking
- Comments/social features
- Notifications

## Setup
Open `index.html` in your browser. No backend required for demo.

## Future Improvements
- Real geocoding/location services
- Persistent storage (database)
- User authentication

- Event filtering and search
