# Personal Finance Tracker

A web app to visualize expenses, set budgets, and track savings goals.

## Features
- Add, view, and delete expenses
- Set and manage budgets
- Track savings goals
- Dashboard with summary
- Data persists in local storage

## How to Use
1. Open `index.html` in your browser.
2. Navigate using the Dashboard, Expenses, Budgets, and Savings tabs.
3. Add your data and see your financial summary.
