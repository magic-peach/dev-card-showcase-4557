# Custom Workout Generator

Build and randomize workout routines based on user goals.

## Features
- Select fitness goals and preferences
- Generate random workout routines
- Save and manage custom workouts
- Dashboard with workout stats
- Settings for customization
- Data persists in local storage

## How to Use
1. Open `index.html` in your browser.
2. Set your goals and generate workouts.
3. Save and manage your routines.

- Add workout history and analytics
