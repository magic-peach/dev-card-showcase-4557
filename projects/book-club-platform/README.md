# Book Club Platform

Create book clubs, schedule meetings, discuss books, and track reading progress.

## Features
- Create or join book clubs
- Schedule meetings and select books
- Discuss books in a chat/discussion area
- Track reading progress for each book
- All data stored in browser (no backend)

## Usage
1. Set your username.
2. Create or join clubs.
3. Schedule meetings and select books.
4. Discuss books and track your reading progress.

## Tech
- HTML, CSS, JavaScript
- All data stored in browser (no backend)
