# Interactive Language Pronunciation Trainer

A web app that helps users practice pronunciation with real-time feedback using speech recognition and audio playback.

## Features
- Enter a word or phrase to practice
- Listen to correct pronunciation
- Speak and get real-time feedback
- View practice history
- See profile info

## Getting Started
Open `index.html` in your browser to use the app.

## Project Structure
- `index.html` — Main HTML file
- `style.css` — Stylesheet
- `app.js` — JavaScript logic
- `README.md` — Project info
