# Local Community Event Finder

A web app that aggregates local events, allows users to RSVP, and connects people with shared interests in their area.

## Features
- Browse upcoming local events
- View event details
- RSVP to events
- See your RSVP history in your profile

## Getting Started
Open `index.html` in your browser to use the app.

## Project Structure
- `index.html` — Main HTML file
- `style.css` — Stylesheet
- `app.js` — JavaScript logic
- `README.md` — Project info
