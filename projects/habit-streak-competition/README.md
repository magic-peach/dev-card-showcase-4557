# Habit Streak Competition

Track daily habits and compete with friends to maintain streaks. Includes leaderboards, reminders, and visualizations of progress to encourage consistency.

## Features
- Add and track daily habits
- Streak tracking for each habit
- Join or create groups to compete
- Leaderboards for group competition
- Progress visualization (charts)
- Daily browser reminders

## Usage
1. Set your username.
2. Add habits you want to track.
3. Mark habits as done each day to maintain streaks.
4. Join or create a group to compete with friends.
5. View your progress and the leaderboard.
6. Enable reminders for daily notifications.

## Tech
- HTML, CSS, JavaScript
- Chart.js for visualization
- LocalStorage for persistence
- Browser Notifications for reminders
