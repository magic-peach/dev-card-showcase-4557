# Digital Art Collaboration Platform

A platform for artists to co-create digital artworks in real time, share feedback, and showcase collaborative projects.

## Features
- Collaborative drawing canvas
- Brush color and size selection
- Save artwork to gallery
- Share feedback on projects
- View collaborative gallery

## Getting Started
Open `index.html` in your browser to use the app.

## Project Structure
- `index.html` — Main HTML file
- `style.css` — Stylesheet
- `app.js` — JavaScript logic
- `README.md` — Project info

## Future Improvements
- Real-time collaboration (WebSocket)
