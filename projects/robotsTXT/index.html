About Robots.txt Files

A robots.txt file tells search engine crawlers which pages or files they can or cannot request from your site.

Important Notes:

  • Robots.txt is a publicly accessible file - anyone can see what you're blocking
  • Not all search engines obey robots.txt rules (malicious crawlers may ignore it)
  • To prevent indexing, use the noindex meta tag or password protection
  • Place your robots.txt file in your website's root directory (e.g., https://example.com/robots.txt)

Common User Agents:

  • * - All search engines
  • Googlebot - Google's web crawler
  • Googlebot-Image - Google's image crawler
  • Bingbot - Microsoft Bing's crawler
  • Slurp - Yahoo's crawler
  • DuckDuckBot - DuckDuckGo's crawler
