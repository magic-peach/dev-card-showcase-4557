# AR Graffiti Wall

A collaborative AR graffiti platform for artists and communities. Create, share, and interact with AR walls anchored to real-world locations.

## Features
- GPS-based AR wall anchors (mocked for web)
- Layered drawing tools
- Public/private walls
- Upvote, comment, and collab mode

## Usage
Open `index.html` in your browser. Create or join a wall, draw, and interact!
