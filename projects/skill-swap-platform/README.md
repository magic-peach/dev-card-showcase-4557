# Skill Swap Platform

Connects users who want to exchange skills (e.g., coding for guitar lessons). Users list skills they offer and seek, chat, and schedule sessions for mutual learning.

## Features
- User profile with skills offered and sought
- Browse/search users by skill
- Chat with other users
- Schedule learning sessions
- All data stored in browser (localStorage)

## Usage
1. Create your profile with skills you offer and seek.
2. Browse users and search by skill.
3. Chat with users to arrange a swap.
4. Schedule sessions for mutual learning.

## Tech
- HTML, CSS, JavaScript
- All data stored in browser (no backend)
