# Recipe Ingredient Substituter

Suggests ingredient swaps for recipes based on dietary restrictions or available pantry items. Users input a recipe, and the app recommends alternatives for missing or unwanted ingredients.

## Features
- Paste recipe ingredients
- Add dietary restrictions (optional)
- Add available pantry items (optional)
- AI suggests substitutes for each ingredient
- Clear, user-friendly results

## Usage
1. Paste your recipe ingredients.
2. Optionally, add dietary restrictions and pantry items.
3. Click "Suggest Substitutes" to get recommendations.

## Tech
- HTML, CSS, JavaScript
- OpenAI API for substitutions
