# Gamified Typing Speed Challenge

Test and improve typing speed with timed challenges, achievements, and a global leaderboard. Fun and competitive.

## Features
- Timed typing challenge
- Scoring (WPM, accuracy)
- Achievements
- Global leaderboard
- Sample leaderboard data

## Setup
Open `index.html` in your browser. No backend required for demo.
