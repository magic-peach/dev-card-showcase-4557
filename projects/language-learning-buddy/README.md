# Language Learning Buddy

Practice vocabulary, grammar, and conversation with AI-powered quizzes and chat.

## Features
- Choose a language and practice area
- AI-generated quizzes (vocabulary, grammar)
- AI-powered conversation practice
- Progress tracking and visualization

## Usage
1. Select a language and practice area.
2. Start a quiz or chat with the AI.
3. Track your progress.

## Tech
- HTML, CSS, JavaScript
- OpenAI API for quizzes and chat
