# Fitness Challenge Hub

Join or create fitness challenges, log workouts, and compete on leaderboards.

## Features
- Join or create fitness challenges (steps, running, cycling, workouts)
- Log workouts and track progress
- Leaderboards for each challenge
- View challenge details, stats, and history
- All data stored in browser (no backend)

## Usage
1. Set your username.
2. Create or join challenges.
3. Log your workouts.
4. View leaderboards and your stats.

## Tech
- HTML, CSS, JavaScript
- All data stored in browser (no backend)
