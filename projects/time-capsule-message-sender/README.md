# Time Capsule Message Sender

Write messages to yourself or others to be delivered via email at a future date. Includes reminders and privacy options.

## Features
- Message input and scheduling
- Email delivery (demo: local storage)
- Reminders
- Privacy options
- History tracking

## Setup
Open `index.html` in your browser. No backend required for demo.
