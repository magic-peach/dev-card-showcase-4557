# Book Recommendation Engine

Suggest books based on user preferences and reading history.

## Features
- Enter your preferences (genre, author, keywords)
- Get personalized book recommendations
- Add books to your reading history
- Beautiful, modern UI
- Settings for customization
- Data persists in local storage

## How to Use
1. Open `index.html` in your browser.
2. Enter your preferences and get recommendations.
3. Add books to your reading history.

## Future Improvements
- Integrate with external book APIs
- Export/import data
- Mobile-friendly design
