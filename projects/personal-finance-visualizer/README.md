# Personal Finance Visualizer

Input income and expenses to see interactive charts, set savings goals, and get budgeting tips. Helps manage finances visually.

## Features
- Income & expense input
- Savings goals
- Interactive charts (Chart.js)
- Budgeting tips
- History tracking

## Setup
Open `index.html` in your browser. No backend required for demo.
