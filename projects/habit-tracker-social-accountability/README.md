# Habit Tracker with Social Accountability

A habit tracker where users can join accountability groups, share progress, and motivate each other to build positive routines.

## Features
- Add and track habits
- Mark habits as completed
- Join or create accountability groups
- View group progress
- See profile info

## Getting Started
Open `index.html` in your browser to use the app.

## Project Structure
- `index.html` — Main HTML file
- `style.css` — Stylesheet
- `app.js` — JavaScript logic
- `README.md` — Project info
