# Virtual Plant Caretaker

Manage a virtual garden, get reminders for real plant care, and learn about botany.

## Features
- Add and manage virtual plants
- Water plants and track last care
- Plant care reminders (browser notifications)
- Botany facts and care tips for each plant
- Remove plants from your garden
- All data stored in browser (no backend)

## Usage
1. Add plants to your virtual garden.
2. Water plants and track care.
3. Enable reminders for plant care notifications.
4. Learn botany facts and tips for each plant.

## Tech
- HTML, CSS, JavaScript
- All data stored in browser (no backend)
