# Virtual Study Group Scheduler

Coordinate study sessions and manage virtual study groups.

## Features
- Create, edit, and delete study groups
- Schedule and manage study sessions
- Calendar view for sessions
- Group member management
- Settings for customization
- Data persists in local storage

## How to Use
1. Open `index.html` in your browser.
2. Create groups and schedule sessions.
3. View your sessions on the calendar.

## Future Improvements
- Add notifications/reminders
- Export/import data
- Mobile-friendly design
