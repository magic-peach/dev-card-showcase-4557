# Eco-Friendly Shopping List

Build shopping lists and see estimated carbon footprints for each item. Suggests greener alternatives and tracks eco-impact.

## Features
- Shopping list input
- Carbon footprint estimation
- Greener alternatives suggestions
- Eco-impact chart (Chart.js)
- Eco tips

## Setup
Open `index.html` in your browser. No backend required for demo.
