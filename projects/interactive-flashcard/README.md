# Interactive Flashcard App

Create, review, and share flashcard decks for learning.

## Features
- Create, edit, and delete flashcard decks
- Add, edit, and delete flashcards in decks
- Review flashcards with flip animation
- Share decks (export/import JSON)
- Settings for customization
- Data persists in local storage

## How to Use
1. Open `index.html` in your browser.
2. Create decks and add flashcards.
3. Review and share your decks.
