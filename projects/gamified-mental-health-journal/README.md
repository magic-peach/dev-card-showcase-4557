# Gamified Mental Health Journal

A journal app that encourages users to log their moods and thoughts, rewarding them with achievements for regular entries.

## Features
- Log mood and thoughts daily
- View journal entries
- Earn achievements for streaks and milestones
- See your profile stats

## Getting Started
Open `index.html` in your browser to use the app.

## Project Structure
- `index.html` — Main HTML file
- `style.css` — Stylesheet
- `app.js` — JavaScript logic
- `README.md` — Project info
